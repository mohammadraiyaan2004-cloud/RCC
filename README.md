# RCC Design Input Validation Tool (IS 456:2000)

A rule-based web application that assists engineers during the **preliminary RCC design stage** by validating key design inputs against **IS 456:2000 durability and detailing provisions**.

This tool focuses on **input validation and decision support**, not full structural design calculations.
